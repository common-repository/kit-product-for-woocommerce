== Description ==
Kit Product For Woocommerce 
This plugin allows to create a single kit of products using multiple products from a single/multiple categories or chosen products.
Allows to create different strategies to boost up sales. Freedom to strategise product sales by combining the products into a kit.


LIVE  DEMO
Visit <a href="https://logicfire.in/dev/product/kit-product-regular-pricing/">live demo</a> to see how plugin works.

FEATURES
New Features will be added soon and also open for suggestions.

#1. Create kits/bundles using multiple products from different categories.
#2. Create kits/bundles from chosen products.
#3. Let Customer create their own kit by selecting products from pre-defined list.
#4. Allow Minimum and Maximum number of products customer can add to a kit.
#5. Add discounts on kit.
#6. Allow freedom to strategise product sales by combining the products into a kit.
#7. Allow customer to update kit even after adding to cart.


== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select “Add New”
3. Search for “Kit Product For Woocommerce”, Install & Activate it
4. Now when you create/edit a product you can choose product type is “Kit Product”, then add products as you want
5. Click Save and see the result
